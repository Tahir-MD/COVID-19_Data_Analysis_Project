# 🦠 SoftGrowTech_COVID-19_Data_Analysis_Project

## 📌 Project Overview

This project is part of my **Data Science Internship at SoftGrowTech**. It involves comprehensive analysis of COVID-19 pandemic data to extract meaningful insights about the global impact of the virus. The analysis covers case trends, death rates, vaccination progress, and regional comparisons using real-time data.

## 🎯 Objectives

- 📊 Analyze global COVID-19 cases, deaths, and recovery trends
- 🌍 Compare pandemic impact across different countries and continents
- 📈 Visualize time series data for daily new cases and deaths
- 💉 Study vaccination progress and its correlation with case reduction
- 🔍 Identify patterns and insights from real-world pandemic data
- 📝 Demonstrate data cleaning, preprocessing, and EDA skills

## 📁 Dataset Information

- **Source:** [Our World in Data (OWID)](https://covid.ourworldindata.org/data/owid-covid-data.csv)
- **Update Frequency:** Daily
- **Time Period:** January 2020 - Present
- **Coverage:** 200+ countries and regions
- **Key Features:**
  - `location`: Country/Region name
  - `date`: Date of observation
  - `total_cases`: Cumulative confirmed cases
  - `new_cases`: Daily new confirmed cases
  - `total_deaths`: Cumulative deaths
  - `new_deaths`: Daily new deaths
  - `total_vaccinations`: Cumulative vaccinations administered
  - `people_vaccinated`: Number of people with at least one dose
  - `people_fully_vaccinated`: Number of fully vaccinated people
  - `population`: Country population
  - `continent`: Continent name
